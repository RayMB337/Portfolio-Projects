## File Shredder (Python)

Built a secure file shredder in Python implementing DoD 5220.22-M sanitization standard with recursive directory support, audit logging, and verified overwrite confirmation

Why: Windows keeps a shadow copy of the files before they are deleted. This project aims to encrypt using professional standards
ensuring private information stays private

## Video Merger (Python)

Built a file merger with python that implements Textualize (TUI toolbox), and ffmpeg via the OS package to merge several videos into one. Textualize provides a professional easy to use TUI that allows the selection of file formats, input video files, and the merged output.

Skills Demonstrated: File IO, FFMPEG mastery, Video Manipulation, TUI development.
